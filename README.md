# Car Brand Classification using Random Forest Classifier

## Overview

This project implements a Machine Learning classification model to predict car brands based on vehicle design and physical characteristics. The project follows the complete ML workflow, including data preprocessing, feature selection, train-test splitting, model training, prediction, and evaluation using the Random Forest Classifier.

## Features

- Data Preprocessing using Label Encoding
- Train-Test Split
- Random Forest Classification
- Prediction
- Accuracy Evaluation
- Classification Report

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Google Colab

## Dataset

Source: YBI Foundation GitHub (MLProjectIdeas Repository)

Dataset: Car Brand Classification for AutoGenius
